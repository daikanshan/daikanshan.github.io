6. 创建渲染器

渲染器是什么，额，你可以把它理解成上帝。所有的角色都已准备就绪，这时就需要上帝这个导演来安排其他人的站位和台词等等。（嘘！其实我们就是那个创造上帝的……）
创建上帝，哦不，是渲染器也很简单。

7. 创造世界！

接下来需要将所有对象放进场景中，并对场景渲染，世界就诞生了！

// 设置cube位置
cube.position.set(0,0,0);

// 设置camera位置和朝向
camera.position.set(-30,40,30);
camera.lookAt(cube.position);

// 设置ambientLight位置
ambientLight.position.set(-40,60,-10);

// 将灯光和3D实体放进scene
scene.add(cube);
scene.add(spotLight);
scene.add(ambientLight);

// 将renderer节点添加到HTML中
document.getElementById("WebGL-output").append(renderer.domElement);

// 渲染
renderer.render(scene,camera);

至此，一个简单的3D场景就创建成功了。总结以下：

  1. threejs将WebGL进行模块化处理，所有的实体都当做对象处理（如场景，灯光，物体，渲染器等）。
  2. threejs将脏活累活都干完了（如对点线面进行渲染，物体的运动等需要大量的几代和高数知识），剩余的创造性的东西留给我们来做。
  3. 距离创造真实的世界还有很长一段路要走。
